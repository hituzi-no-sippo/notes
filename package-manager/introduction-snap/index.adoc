link:{snap-url}[snap]はapp packageです。次の特徴を持っています。

.snapの特徴
* 他のpackageとの依存がない +
  packageごとに、upgrade, downgradeできる。
* Linuxのdistributionが関係ない +
  Ubuntu, Arch Linux, Kali Linuxなどで使えます。
  link:{installing-snapd-linux-url}[対応しているdistribution list^]
* 自動更新がある +
  metadateやpackageの自動更新があります。更新頻度は設定できます。
